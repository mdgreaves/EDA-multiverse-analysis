Multiverse analysis of electrodermal activity-estimated fear learning differences between anxious and non-anxious individuals
-----------------------------------------------------------------------------------------------------------------------------
Conditioned response estimates, expectancy ratings, and original code associated with Greaves et al. For Study 1, the self-report and demographic data matrix includes participant IDs; scores on the anxiety scale from the short-form Depression Anxiety Stress Scales; sex (1 = female); and age. Details of the self-report and demographic data used in Study 2, are stored in the relevant table's *VariableNames* property (*ClinStatus* = 1 for spider phobia). With regards to MAT-files containing conditioned response estimates, each "model" structure contains:
>.CR [nTrials x 1]: conditioned responses;</br>
.CS [nTrials x nCues]: Logical matrix for conditioned stimuli; and,</br>
.US [nTrials x 1]: Logical vector for unconditioned responses.
